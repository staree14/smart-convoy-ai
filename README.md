# SmartConvoy AI - Military Convoy Management System

A React + Tailwind CSS dashboard for managing military convoy routes and operations.

## Features

- **Sidebar Navigation**: Dark navy sidebar with menu items (Dashboard, Plan Routes, Active Convoys, Analytics, Settings)
- **Route Planning Form**: Source/Destination inputs, Priority dropdown, Vehicle type selection
- **Interactive Map**: Visual representation of convoy routes with colored lines (green=optimized, yellow=conflict), truck icons, and checkpoint markers
- **Convoy Queue Panel**: Scrollable list of convoy cards with priority badges, status indicators, and merge suggestions
- **Metrics Dashboard**: Real-time metrics showing active convoys, fuel savings, merged convoys, and conflicts

## Installation

```bash
npm install
```

## Running the Application

```bash
npm start
```

The application will open at `http://localhost:3000`

## Technologies

- React 18
- Tailwind CSS 3
